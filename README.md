# 🎓 CGPA Calculator Web Application

## 📌 Overview
This project is a responsive web application designed to calculate CGPA (Cumulative Grade Point Average) based on subject grades and credits. It provides real-time results and supports multi-semester tracking.

---

## 🚀 Features
- 📊 Credit-based CGPA calculation
- 📈 Multi-semester performance tracking
- ⚡ Real-time result updates
- 📄 CGPA to Percentage conversion
- 📱 Responsive design (mobile + desktop)
- 📥 PDF report generation

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

 🧠 How It Works
CGPA is calculated using the formula:

CGPA = Σ(Grade × Credit) / Total Credits

The application dynamically updates results as the user inputs grades and credits.
